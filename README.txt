Black Tiger Computing Pages Starter v2

Files included:
- index.html
- styles.css
- .nojekyll

How to publish:
1. Create or open the repository named black-tiger-computing.github.io
2. Upload these files to the root of that repository
3. Go to Settings -> Pages
4. Set Build and deployment to:
   - Deploy from a branch
   - main
   - /(root)
5. Save

Your live URL will be:
https://black-tiger-computing.github.io

Update these placeholders after publishing:
- LinkedIn link
- email address
- Ritual repo link if the slug is different
